Resumo para retomada do projeto de predição de evasão universitária:

1. Estrutura principal:
- Scripts principais: full-experiments/main.py e periods-experiments/main.py
- Notebooks de análise exploratória: exploratory-analysis/
- Dados necessários: arquivos .csv (não estão no repositório)

2. Passos para rodar:
a) Obter os arquivos de dados (ex: data-generator/newest-data-base/full-dv-data.csv)
b) Criar ambiente virtual: python -m venv venv
c) Ativar ambiente: venv\Scripts\activate
d) Instalar dependências: pip install pandas numpy matplotlib seaborn scikit-learn
e) Rodar experimentos: python main.py nas pastas full-experiments ou periods-experiments

3. Funcionamento geral:
- Entrada: dados dos alunos (csv)
- Processamento: análise, pré-processamento e experimentos de machine learning
- Saída: resultados em arquivos e gráficos

Observação: Sem os arquivos de dados, os scripts não funcionarão corretamente.
